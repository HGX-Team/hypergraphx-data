pokemon-moves-version-group-multiplex

Tags:

Multiplex Biology Games

Description

Multiplex move learnset hypergraph with PokéAPI Pokémon entries as nodes and game version groups as layers. In each layer, a hyperedge corresponds to one move and contains the Pokémon that can learn it in that version group.
