coauth-PACS-2

Tags:

Undirected Social Authorship

Description

Co-authorship hypergraph built from physics publications indexed by the Physics and Astronomy Classification Scheme (PACS). Nodes represent authors. Each hyperedge represents a publication and contains its set of co-authors. This subset includes publications whose PACS code belongs to top-level category 2 (based on the first digit of the PACS code).
