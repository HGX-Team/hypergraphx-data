coauth-geology

Tags:

Undirected Temporal Social Authorship

Description

Nodes represent authors. Each hyperedge represents a publication and contains the set of co-authors of that publication. This is a temporal higher-order network dataset, where each hyperedge is a publication marked with the Geology tag in the Microsoft Academic Graph. Timestamps are the year of publication.
