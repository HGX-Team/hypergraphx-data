cooking-multiplex

Tags:

Multiplex Food

Description

Multiplex recipe hypergraph spanning multiple cuisines. Nodes represent ingredients. Each hyperedge represents a recipe and contains the set of ingredients used in that recipe. Layers correspond to different cuisine collections.
