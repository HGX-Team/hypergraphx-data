crypto-bitcoin-201506

Tags:

Directed Temporal Finance

Description

Nodes represent Bitcoin addresses. Each hyperedge represents a transaction and contains the set of addresses involved in that transaction. This dataset contains the first 1.5M Bitcoin transactions of Jun. 2015.
