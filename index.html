Quick ways to load datasets

Minimal snippets to get data into your workflow.

Download & read 
from hypergraphx.readwrite import load_hypergraph
H = load_hypergraph("DATASET_PATH")
Replace DATASET_PATH with the file path.
Dataset loader 
from hypergraphx.readwrite import load_hypergraph_from_server
H = load_hypergraph_from_server("DATASET_NAME")
Replace DATASET_NAME with a dataset identifier.
HIF 
from hypergraphx.readwrite import write_hif
write_hif(H, "DATASET_NAME.hif.json")
Replace DATASET_NAME.hif.json with the file path.
