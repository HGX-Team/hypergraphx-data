# email-Enron-core reproducibility

## Dataset
- Name: email-Enron-core
- Description: Nodes represent email addresses. Each hyperedge represents an email message, connecting the sender with all recipients (and may include a timestamp/weight when available). This dataset contains the emails generated by core employees of the Enron company.
