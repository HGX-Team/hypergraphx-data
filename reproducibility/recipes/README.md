# recipes reproducibility

## Dataset
- Name: recipes
- Description: A hypergraph representing the relationships between different flavors and ingredients in culinary recipes. Nodes represent ingredients, and hyperedges represent recipes that combine multiple ingredients. Hyperedges have associated metadata tags indicating the type of cuisine by geography (e.g., African, East-Asian, Western European).
