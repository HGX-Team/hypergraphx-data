# ECO-M_PL_062_plants reproducibility

## Dataset
- Name: ECO-M_PL_062_plants
- Description: Nodes represent plant species. Each hyperedge represents a pollinator and contains the set of plant species visited by that pollinator (projection of a plant–pollinator bipartite network). This dataset contains the relationships between plants and pollinators from the 'Web of Life' project.

## Source data
- https://www.web-of-life.es/
