# language-country reproducibility

## Dataset
- Name: language-country
- Description: A hypergraph describing which languages are spoken in which countries. Nodes represent countries and hyperedges represent languages spoken by the countries.

## Source data
- University of Groningen. 2016. "World Languages." University of Groningen Open Data. http://opendata.rug.nl/data sets/5c6ec52c374249a781aede5802994c95_0?geometry=153.457%2C15.653%2C-122.52%2C76.276. Accessed through Resource Watch, (date). www.resourcewatch.org.
