# ECO-M_PL_062_pollinators reproducibility

## Dataset
- Name: ECO-M_PL_062_pollinators
- Description: Nodes represent pollinator species. Each hyperedge represents a plant species and contains the set of pollinators visiting that plant (projection of a plant–pollinator bipartite network). This dataset contains the relationships between plants and pollinators from the 'Web of Life' project.
