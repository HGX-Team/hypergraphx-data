# flavourcompounds reproducibility

## Dataset
- Name: flavourcompounds
- Description: A hypergraph representing the relationships between ingredients and their chemical compounds in various recipes. Each hyperedge connects multiple compounds that are found in a single ingredient. Hyperedges (ingredients) have metadata including ingredient name and category (Flower, Fish, Herb, etc.). Nodes (compounds) have metadata including compound name and CAS number.
